Heroes on the Move is an application designed to aid military families in gathering important information and recommendations upon each move that they have to make. All code for the Lambda functions along with a specific README can be found in the folder labeled Lambda Functions Code. All code for the Frontend along with a specific README can be found in the folder labeled S3 Frontend Code. Finally, the Swagger file used for the API along with a specific README can be found in the the folder labeled APIGateway Code.
